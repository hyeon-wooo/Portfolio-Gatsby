TECHNOLOGY

보유 기술

HTML5, CSS3

  • media query, flex box를 이용한 반응형 웹 제작 가능

Javascript, Typescript

  • "주력 언어"
  • 코어 객체 메서드 사용 숙달
  • 실행 컨텍스트, 스코프, 프로토타입 등 동작 원리 숙지
  • ES6+ 문법 및 비동기 처리 숙달

React

  • 함수형 컴포넌트 및 Hooks 사용 가능
  • 컴포넌트 생애 주기 숙지
  • Ajax 활용 가능

React Native

  • "주력 스킬"
  • 자율적으로 필요한 라이브러리 검색 및 적용 가능
  • 필요한 경우 간단한 라이브러리는 분석 및 튜닝
  • Navigation 사용 숙달
  • Ajax, GraphQL을 이용한 통신
  • Google Play, App Store에 어플 배포 경험
  • Animation 적용 및 제어
  • 함수형 컴포넌트 작성 숙달, 클래스형 컴포넌트 사용 가능

GraphQL

  • 서버 단에서 리졸버와 쿼리 작성 가능
  • variables 활용

Gatsby.js

  • React와 Graphql을 기반으로 하는 Gatsby.js를 이용하여 정적사이트 생성 가능
  • 본 사이트가 Gatsby로 제작한 사이트입니다

Redux

  • 전역 State 생성 및 Action 적용
  • 특정 컴포넌트에서 전역 State 변화에 대응하여 re-rendering 가능

Node.js

  • Express 프레임워크를 이용하여 서버 구축 가능
  • Sequelize ORM 라이브러리를 이용한 MySQL(MariaDB) 연동 및 쿼리 관리
  • Json Web Token을 이용한 사용자 인증

Git & GitHub

  • Stage 관리, 커밋 생성 및 체크아웃
  • 원격 저장소에 Push, Pull
  • Branch 생성, 전환 및 병합

Unity 3D

  • 서버를 두어 연동하는 수준은 아님
  • 카메라 제어 및 모델의 중력 질량 운동 등을 적용하며 자유로운 스크립트 작성과 Scene 전환이 가능한 수준

MariaDB, MySQL

  • 데이터 CRUD, 서브쿼리, JOIN, 외래키 관리, 뷰, 인덱스 관리 등 기본 기능들 숙달
  • 파티셔닝, 프로시저, 트리거, 동시성 제어에 관한 개념 숙지

MongoDB

  • 데이터 CRUD가 가능한 기초적인 수준
